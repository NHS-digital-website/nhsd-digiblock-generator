The main scope of this project is to streamline traditional UI stack (HTML + CSS + JS) workflows and incorporate best practices for UI development when building NHSUK or GOVUK digital services.

This project uses Gulp 4, Webpack 4, Node SASS, Babel to build production-ready front-end assets using the NHSUK or GOVUK front-end toolkit - or no toolkit at all if you need a blank canvas for your project.

Find out more about this project

This is not the official NHSUK prototyping toolkit.

You can use this project to build or prototype a NHSUK service, but it is important to note that this is not the official NHSUK prototyping kit, and it is not maintained by GDS.
